Open Source Trajectory Extraction Project for Anisotropic 3D Printing

During my internship at Inria, I worked on a project focused on trajectory extraction for anisotropic 3D printing. The main objective was to translate existing Python code into C++, sourced from a research paper. This project involved creating a C++ library containing the necessary functionalities to generate trajectories suited for anisotropic 3D printing. The library was designed to extract cycles from a scalar field representing printing trajectories. These cycles, optimized to follow anisotropic variations, ensure high-quality printing results. The project is open source and available on my GitHub, providing the community with the opportunity to contribute and benefit from these functionalities for their own 3D printing projects.
